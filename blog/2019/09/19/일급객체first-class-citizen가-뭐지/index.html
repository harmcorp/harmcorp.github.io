    일급객체(First class citizen)가 뭐지?

    1급 객체란? 아래 3 가지조건을 충족한다면 1급 객체라고 할수 있습니다.

    1. 변수나 데이타에 할당 할 수 있어야 한다.
    2. 객체의 인자로 넘길 수 있어야 한다.
    3. 객체의 리턴값으로 리턴 할수 있어야 한다.

    javascript 에서는 함수가 1급 객체지만, java 에선 그렇지 않다.
